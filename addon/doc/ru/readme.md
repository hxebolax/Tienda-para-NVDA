# TiendaNVDA )Магазин дополнений для NVDA)

> **⚠️ Важное уведомление для тестировщиков бета-версии:**
> Если вы тестировали дополнение **TiendaNVDA_Modern**, пожалуйста, **удалите его перед установкой этой версии**. Та версия была тестовой и бета-версией, которая не должна сосуществовать с этой финальной версией. Чтобы удалить его, перейдите в меню NVDA → Сервис → Магазин дополнений, выберите "TiendaNVDA_Modern" и удалите его. Перезапустите NVDA, после чего приступайте к установке этой новой версии.

Единый Магазин дополнений для NVDA: интегрирует **Магазин испаноязычного сообщества (NVDA.ES)** и **Официальный магазин NV Access** в единый доступный интерфейс.

<a name="introduccion"></a>
## Введение

**Магазин дополнений для NVDA** — это полная эволюция старого Магазина для NVDA.ES, переработанная с нуля для обеспечения современного, быстрого и унифицированного интерфейса.

### Что нового по сравнению с предыдущей версией?

- **Единый магазин:** Просматривайте дополнения NVDA.ES и официального магазина NV Access в одном окне.
- **Индикаторы состояния:** Каждое дополнение показывает свой статус в реальном времени: установлено, доступно обновление, отключено, несовместимо и т.д.
- **Локальное управление:** Отключайте, включайте или удаляйте дополнения, не выходя из магазина.
- **Многоуровневая система кэширования:** Кэш серверов, кэш переводов и кэш списков для сверхбыстрой загрузки.
- **Офлайн-режим:** Просматривайте магазин без подключения к интернету, используя сохраненные данные кэша.
- **Резервное копирование и восстановление:** Создавайте резервные копии ваших дополнений и восстанавливайте их при смене компьютера.
- **Упаковщик:** Создавайте файлы `.nvda-addon` из любого установленного дополнения.
- **Тихая установка:** Устанавливайте дополнения в фоновом режиме без промежуточных диалогов.
- **Умная перезагрузка:** Магазин определяет, было ли что-то действительно установлено, прежде чем предложить перезагрузку.
- **Проверка зависимостей:** Проверяет, удовлетворены ли зависимости дополнения, перед установкой.
- **Перевод с кэшированием:** Мгновенно переводите описания с помощью клавиши F3, причем переводы сохраняются, чтобы не повторять запрос.
- **Улучшенные уведомления:** Уведомления об обновлениях теперь указывают источник (NVDA.ES или официальный) и названия дополнений.

<a name="instalacion"></a>
## Установка

1. Скачайте файл `.nvda-addon` со страницы [релизов репозитория](https://github.com/hxebolax/Tienda-para-NVDA/releases).
2. Откройте скачанный файл или перетащите его на окно NVDA.
3. Подтвердите установку, когда NVDA запросит это.
4. Перезапустите NVDA для активации дополнения.

<a name="primeros-pasos"></a>
## Первые шаги

Дополнение поставляется **без назначенных сочетаний клавиш**. Вы можете назначить свои сочетания в:

**Меню NVDA → Параметры → Жесты ввода → Магазин дополнений NVDA**

Здесь вы найдете следующие доступные действия:

- Показать окно со всеми дополнениями NVDA.ES
- Проверить обновления установленных дополнений в NVDA.ES
- Показать окно со всеми дополнениями официального магазина
- Проверить обновления официальных дополнений
- Показать единый магазин со всеми источниками дополнений

### Доступ через меню

Вы также можете получить доступ ко всем функциям через меню NVDA:

**Меню NVDA → Сервис → Магазин дополнений NVDA**

Здесь вы найдете следующие подменю:

- **Магазин NVDA.ES:** Список дополнений и поиск обновлений испаноязычного сообщества.
- **Официальный магазин NVDA:** Список дополнений и поиск обновлений официального магазина.
- **Единый магазин (Все источники):** Показывает все дополнения из всех источников в одном списке.
- **Упаковщик дополнений:** Позволяет упаковать установленные дополнения в файлы `.nvda-addon`.
- **Документация дополнения:** Открывает эту документацию в вашем браузере по умолчанию.

<a name="las-tres-tiendas"></a>
## Три магазина

Новая версия интегрирует три режима просмотра дополнений:

### Магазин NVDA.ES

Это магазин испаноязычного сообщества. Он получает дополнения с сервера [https://nvda.es](https://nvda.es) и любых добавленных вами пользовательских серверов.

### Официальный магазин NVDA

Предоставляет доступ к официальному магазину дополнений NV Access ([https://addons.nvda-project.org](https://addons.nvda-project.org)). Дополнения из этого источника получаются напрямую через API официального магазина и отображаются со всей информацией о совместимости.

### Единый магазин

Это комбинированное представление, которое показывает **все дополнения из всех источников** в одном списке. Дополнения помечаются тегами `[ES]` (испаноязычное сообщество) и `[OF]` (официальный магазин), чтобы вы знали происхождение каждого.

<a name="la-interfaz-de-la-tienda"></a>
## Интерфейс магазина

При открытии любого из магазинов отображается окно, разделенное на две панели:

### Левая панель (рабочая область)

1. **Поле поиска:** При открытии магазина фокус находится здесь. Введите любой термин и нажмите Enter, чтобы отфильтровать список. Чтобы снова показать все дополнения, очистите поле поиска и нажмите Enter с пустым полем.

2. **Список дополнений:** Показывает все доступные дополнения с их индикатором состояния в квадратных скобках (например: `[I]`, `[U]`). Навигация клавишами Вверх/Вниз.

3. **Кнопка действия (Установить/Обновить):** Эта кнопка динамическая; её текст автоматически меняется в зависимости от состояния выбранного дополнения:
   - Если дополнение у вас не установлено, отображается **"Установить"**.
   - Если доступно обновление, отображается **"Обновить"**.

### Правая панель (информационная вкладка)

По мере перемещения по списку дополнений эта панель заполняется полной информацией о выбранном дополнении:

- Имя и краткое описание
- Версия, доступная на сервере
- Установленная версия (если применимо)
- Автор
- Полное описание
- Совместимость с NVDA (минимальная версия и последняя протестированная)
- Количество загрузок (когда доступно)
- Состояние установки

<a name="indicadores-de-estado"></a>
## Индикаторы состояния

При перемещении по списку дополнений NVDA объявляет буквы в квадратных скобках, указывающие на состояние каждого дополнения:

| Индикатор | Значение |
|:----------|:------------|
| **[I]** | **Установлено:** Дополнение установлено и активно. |
| **[U]** | **Обновление:** Доступна новая версия. Обновитесь! |
| **[U-I]** | **Несовместимое обновление:** Доступна новая версия, но она несовместима с вашей версией NVDA. |
| **[D]** | **Отключено:** Дополнение установлено, но деактивировано вручную. |
| **[R]** | **Ожидает удаления:** Будет удалено при перезапуске NVDA. |
| **[I-I]** | **Установлено несовместимое:** Дополнение установлено, но заблокировано из-за несовместимости с вашей версией NVDA. |
| **[X]** | **Несовместимо:** Дополнение несовместимо с вашей версией NVDA. |

В **Едином магазине** также отображается происхождение:

| Тег | Происхождение |
|:---------|:------------|
| **[ES]** | Серверы NVDA.ES (испаноязычное сообщество) |
| **[OF]** | Официальный магазин NV Access |

<a name="teclas-rapidas-y-funciones-especiales"></a>
## Горячие клавиши и специальные функции

Эти клавиши работают, когда фокус находится в списке дополнений:

### F1 — Текущая позиция

Нажмите **F1**, чтобы NVDA сообщил вам, на какой позиции в списке вы находитесь: "Вы на дополнении 15 из 200".

### Ctrl+F1 — Объяснить индикатор

Не помните, что означают скобки `[I]` или `[U]`? Нажмите **Ctrl+F1**, и NVDA понятным языком объяснит состояние выбранного дополнения.

### F2 — Прочитать полную карточку

Нажмите **F2**, чтобы NVDA одним разом прочитал всю техническую карточку и описание дополнения **без необходимости переходить на правую панель**. Работает во всех магазинах.

### F3 — Перевести описание

Описание на английском или другом языке? Нажмите **F3**, и магазин переведет его на ваш настроенный язык (по умолчанию — испанский). При начале и окончании перевода будет воспроизведен звуковой сигнал.

> **Примечание:** Для использования F3 необходимо предварительно активировать переводчик в опциях дополнения. Требуется подключение к интернету.

<a name="menu-contextual"></a>
## Контекстное меню

В списке дополнений нажмите **Клавишу приложений** (или **Shift+F10**), чтобы открыть контекстное меню со следующими опциями:

### Фильтры

- **Показать все дополнения:** Показывает полный список (опция по умолчанию).
- **Показать дополнения по совместимости API:** Фильтрует только дополнения, совместимые с конкретной версией NVDA.
- **Показать дополнения, отсортированные по автору:** Сортирует список по имени автора.
- **Показать по убыванию загрузок:** Сортирует по популярности.

> **Примечание:** Фильтры не суммируются. Каждый фильтр применяется индивидуально, и заголовок окна меняется, чтобы сообщить об активном фильтре. Опции фильтра сохраняются, пока NVDA не перезапустят.

### Копировать в буфер обмена

- **Копировать информацию:** Копирует полную карточку выбранного дополнения.
- **Копировать ссылку на веб-страницу:** Копирует официальный URL дополнения.
- **Копировать ссылку для скачивания:** Подменю с доступными ветками разработки для копирования их прямой ссылки на скачивание.

<a name="gestion-de-complementos-instalados"></a>
## Управление установленными дополнениями

Одно из самых мощных нововведений: если дополнение уже установлено, вы можете управлять им **не выходя из магазина**.

1. Выберите дополнение, помеченное как `[I]`, `[D]` или `[U]` в списке.
2. Нажмите **Клавишу приложений** (или правую кнопку мыши).
3. В подменю **Управление установленным** вы найдете:

- **Отключить / Включить:** Временно активирует или деактивирует дополнение.
- **Удалить:** Помечает дополнение на удаление (вступит в силу после перезапуска NVDA).
- **Просмотреть документацию:** Открывает документацию дополнения в браузере. Если документация есть на вашем языке, она откроется на вашем языке; в противном случае — на языке дополнения по умолчанию.

<a name="empaquetador-de-complementos"></a>
## Упаковщик дополнений

Упаковщик позволяет создавать файлы `.nvda-addon` из уже установленных у вас дополнений. Он идеально подходит для:

- Передачи дополнения другому человеку без необходимости искать его в магазине.
- Создания резервных копий конкретных дополнений.
- Сохранения определенной версии дополнения перед её обновлением.

**Чтобы упаковать дополнение:**

1. Перейдите в **Меню NVDA → Сервис → Магазин дополнений NVDA → Упаковщик дополнений**.
2. Выберите дополнение, которое хотите упаковать, из списка.
3. Выберите каталог, в который хотите сохранить файл.
4. Файл `.nvda-addon` будет автоматически создан в формате: `имя_версия_Gen.nvda-addon`.

<a name="buscar-actualizaciones"></a>
## Поиск обновлений

Дополнение предлагает два способа поиска обновлений:

### Ручной поиск

В меню Сервис выберите **Проверить обновления** в любом из магазинов (NVDA.ES или официальном). Отобразится окно с дополнениями, для которых доступны обновления.

В этом окне вы можете:

- **Выбирать дополнения по отдельности:** Используйте пробел для отметки/снятия отметки.
- **Alt+S:** Выбрать все дополнения для обновления.
- **Alt+D:** Снять выделение со всех дополнений.
- **Alt+A:** Начать обновление выбранных дополнений.
- **Alt+C / Escape / Alt+F4:** Закрыть окно.

### Автоматическая проверка

Когда вы активируете автоматическую проверку в опциях:

- Магазин будет искать обновления в фоновом режиме с заданным интервалом.
- Будет отображаться системное уведомление с указанием количества обновлений и их источника.
- Поиск автоматически останавливается после 10 проверок без результатов или 5 проверок после обнаружения обновлений, чтобы не перегружать сервер.

Теперь уведомления стали более информативными:

```
Найдено 3 обновления.
• 
NVDA.ES (2): Дополнение A, Дополнение B
• 
Официальный магазин (1): Дополнение C
Запустите проверку обновлений дополнений.
```

<a name="panel-de-opciones"></a>
## Панель опций

Доступ к настройкам дополнения осуществляется через:

**Меню NVDA → Параметры → Опции → Магазин дополнений NVDA**

### A. Магазин NVDA.ES

- **Выберите сервер дополнений:** Выбирает сервер по умолчанию из числа настроенных.
- **Управление серверами дополнений:** Открывает менеджер, где вы можете добавлять, редактировать или удалять пользовательские серверы.

### B. Официальный магазин NVDA

- **Включить официальный магазин NVDA:** Активирует или деактивирует интеграцию с официальным магазином NV Access.
- **Разрешить несовместимые дополнения из официального магазина:** Позволяет пытаться устанавливать дополнения, помеченные как несовместимые. **Используйте на свой страх и риск.**

### C. Обновления

- **Активировать автоматическую проверку обновлений:** Включает фоновый поиск.
- **Время проверки обновлений:** Выбирает интервал между проверками:
  - 15 минут, 30 минут, 45 минут, 1 час, 12 часов, 1 день, 1 неделя.
- **Включить обновления из официального магазина:** Добавляет обновления из официального магазина в автоматическую проверку.

### D. Перевод

- **Активировать переводчик для описаний:** Включает использование клавиши F3 для перевода.
- **Язык для перевода описаний:** Позволяет выбрать из 12 языков: немецкий, арабский, хорватский, испанский, французский, английский, итальянский, польский, португальский, русский, турецкий и украинский.

### E. Общие опции

- **Сортировать дополнения по алфавиту:** Сортирует список от A до Я.
- **Устанавливать дополнения после скачивания:** Автоматически открывает мастер установки по завершении загрузки.
- **Тихая установка:** Дополнения устанавливаются в фоновом режиме без промежуточных диалогов. Только запрашивает перезагрузку по окончании.
- **Включить кэш серверов:** Сохраняет списки дополнений на диск для более быстрой загрузки.
- **Обновлять кэш каждые...:** Настраивает интервал обновления кэша.
- **Использовать кэш для переводов:** Переводы, сделанные с помощью F3, сохраняются, чтобы не повторять запрос к Google.
- **Включить офлайн-режим:** Позволяет просматривать магазин без подключения к интернету, используя сохраненные в кэше данные.

### F. Резервное копирование и восстановление

- **Создать резервную копию дополнений:** Создает файл JSON со списком всех ваших установленных дополнений.
- **Восстановить из резервной копии:** Загружает файл резервной копии и позволяет переустановить перечисленные дополнения.

### G. Установленные дополнения, присутствующие на сервере

В нижней части опций отображается список ваших дополнений, которые также есть на сервере. Отсюда вы можете:

1. Выбрать дополнение и нажать **Пробел**.
2. Во всплывающем меню выбрать **канал обновления** (Стабильный, Бета, Разработка и т.д.) или **Отклонить обновления**, чтобы магазин перестал уведомлять об этом дополнении.

> **Важно:** Изменения сохраняются только при нажатии ОК или Применить в диалоге опций.

<a name="sistema-de-cache"></a>
## Система кэширования

Магазин реализует многоуровневую систему кэширования для максимальной производительности:

### Кэш серверов

Сохраняет списки дополнений на диск. При открытии магазина, если кэш не истек, он загружается непосредственно с диска вместо отправки запроса на сервер.

- Настраивается через: **Включить кэш серверов** в опциях.
- Интервал обновления настраивается.

### Кэш переводов

Переводы, выполненные с помощью F3, сохраняются в постоянный файл JSON. При следующем запросе того же перевода он загрузится мгновенно из кэша.

- Настраивается через: **Использовать кэш для переводов** в опциях.

### Кэш в памяти

Помимо дискового кэша, магазин поддерживает кэш в оперативной памяти для наиболее частых запросов, полностью исключая обращение к диску во время сессии.

<a name="modo-offline"></a>
## Офлайн-режим

Офлайн-режим позволяет просматривать магазин **без подключения к интернету**, используя данные, предварительно сохраненные в кэше.

Для его использования:

1. Убедитесь, что опции **Включить кэш серверов** и **Включить офлайн-режим** активированы в настройках.
2. Просмотрите магазин хотя бы один раз при подключении к интернету, чтобы сформировался кэш.
3. В следующий раз, когда вы откроете магазин без интернета, данные загрузятся из кэша.

> **Примечание:** В офлайн-режиме вы не сможете скачивать или устанавливать дополнения, но сможете просматривать информацию о дополнениях, которые посещали ранее.

<a name="backup-y-restauracion"></a>
## Резервное копирование и восстановление

### Создание резервной копии

1. Перейдите в **Опции → Магазин дополнений NVDA → Создать резервную копию дополнений**.
2. Выберите имя и расположение для файла `.json`.
3. Будет создан файл со списком всех ваших установленных дополнений, включая имя, версию и краткое описание.

### Автоматическое резервное копирование при выходе

Магазин автоматически создает резервную копию при закрытии NVDA (настраивается в опциях).

### Восстановление из резервной копии

1. Перейдите в **Опции → Магазин дополнений NVDA → Восстановить из резервной копии**.
2. Выберите файл `.json` резервной копии.
3. Магазин запустит мастер, который найдет последние версии каждого дополнения на серверах и позволит установить их пакетно.

> **Идеально подходит для:** Переноса дополнений на новый компьютер или восстановления вашей конфигурации после переустановки NVDA.

<a name="traduccion-de-descripciones"></a>
## Перевод описаний

Магазин включает встроенный переводчик на основе Google Translate:

1. **Активируйте переводчик** в опциях дополнения.
2. **Выберите целевой язык** (по умолчанию испанский).
3. **Нажмите F3** на любом дополнении в списке, чтобы перевести его описание.

Особенности:

- Звуковой сигнал в начале и в конце, указывающий на выполнение перевода.
- Переводы сохраняются в кэш, чтобы не повторять запросы.
- Перевод теряется при смене дополнения; нажмите F3 снова, если он вам понадобится.
- Требуется подключение к интернету.

<a name="servidores-personalizados"></a>
## Пользовательские серверы

Вы можете добавлять сторонние репозитории дополнений, использующие формат, совместимый с NVDA.ES.

### Добавление сервера

1. Перейдите в **Опции → Магазин дополнений NVDA → Управление серверами дополнений**.
2. Нажмите **Добавить**.
3. Введите описательное имя и URL сервера.
4. Подтвердите, и сервер появится в списке выбора серверов.

### Пример: Сервер русскоязычного сообщества

- **Имя:** Русскоязычное сообщество
- **URL:** `https://nvda-addons.ru/get.php?addonslist`

### Быстрая смена сервера

Из главного окна магазина NVDA.ES нажмите **Alt+C** или кнопку **Сменить сервер**, чтобы открыть меню со всеми настроенными серверами. Смена происходит мгновенно и временно (не сохраняется как стандартная, пока вы не измените её в опциях).

> **Примечание:** Сервер по умолчанию для испаноязычного сообщества не может быть изменен или удален.

<a name="observaciones-y-protecciones"></a>
## Примечания и защиты

Дополнение включает множество защит для обеспечения безопасного использования:

1. **Дополнения, ожидающие удаления:** Автоматически исключаются из проверок обновлений.
2. **Проверка совместимости API:** Даже если версия на сервере новее, но она несовместима с вашей версией NVDA, обновление не будет предложено.
3. **Уведомление об ошибках установки:** Если какое-либо дополнение не удалось обновить, сообщается его имя.
4. **Блокировка после обновления:** Магазин не позволяет искать дальнейшие обновления, если обновление уже было выполнено, а NVDA не перезапущен.
5. **Уведомление после перезагрузки:** Если автоматическая проверка обнаруживает, что NVDA не был перезапущен после обновления, выдается напоминающее уведомление.
6. **Защита без интернета:** Если библиотеки не могут быть загружены из-за отсутствия соединения, информация записывается в журнал NVDA и выдается голосовое сообщение при попытке доступа к магазину.
7. **Умная перезагрузка:** Магазин автоматически определяет, было ли действительно установлено дополнение. Если вы отмените установку, он не будет просить перезагрузку без необходимости.
8. **Проверка зависимостей:** Перед установкой проверяет, удовлетворены ли все требуемые зависимости.
